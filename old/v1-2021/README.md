# Pyramid v1.0 (2021)

The original Pyramid release — a simple PureCloud (Genesys Cloud) user/queue exporter distributed as a standalone `.exe`.

## Files

| File | Description |
|------|-------------|
| `pyramid.jpg` | Original screenshot of the v1.0 GUI |
| `.gitattributes` | Git LFS config (the `.exe` was tracked via LFS) |

## Notes

- The original `pyramid.exe` (30 MB, PyInstaller-packed) is **not preserved here** — it was a compiled binary with no source separation.
- The v1.0 source code evolved directly into the current `src/pyramid.py`.
- The original README described it as: *"This program allows you to download in .csv format all users and queues from purecloud instance"*
